Este manual cubre el proceso completo para desplegar la base de datos GradosTitulos en Microsoft Azure. Se parte desde la creación de la cuenta con el correo institucional T00050A@ms.upla.edu.pe, se configura un servidor SQL en Azure, y finalmente se ejecutan los scripts de creación de esquemas, 25 tablas, vistas, funciones y stored procedures.
